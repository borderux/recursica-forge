Add Time Picker component with adapter implementations for Mantine, Material UI, and Carbon. Includes a time input field with a leading clock icon that opens a popover for selecting hours and minutes, and an AM/PM period selector using the Dropdown component with CSS variable overrides to share TimePicker styling (border size, border color, border radius, background color, text color, vertical padding, and text style). Adds TimePicker preview, toolbar configuration, UIKit token definitions, and component registry entries across all three shells.
